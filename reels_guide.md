# FinLab — Reels / short-video SCRIPT guide

**Read `BRAND.md` in this repo first — positioning, voice formula, and hard bans live there and override anything below.**

This produces a **script + on-screen text + caption + hashtags + shot ideas** from a topic.
It does NOT produce a finished video — Olesya films and edits it herself. Write the plan she
can shoot.

## Voice (non-negotiable, same as everywhere)
- Olesya's voice: **zero dashes (—), zero negations** (frame everything affirmatively), calm,
  precise, warm. Russian.
- People should recognize themselves in every line. Quiet accuracy, not hype.
- Consult `references/marketing-tiktok-strategist.md` and
  `references/marketing-video-optimization-specialist.md` for hook craft and retention — but
  keep FinLab's calm tone, never generic hype-bro energy.

## Structure (15–30 seconds)
1. **Hook (first 3 seconds)** — one line that stops the scroll. A recognizable feeling, a
   quiet claim, or a precise question. This is the single most important line.
2. **3–5 beats** — each a short spoken line that delivers one idea. Keep momentum; no dead air.
3. **Payoff** — one clear takeaway the viewer leaves with.
4. **Soft CTA** — one line (follow, try the calculator/test/session, as relevant).

## Deliverable format (return exactly this shape)
```
🎬 Reels-сценарий: <тема>

ХУК (0–3 сек): <одна строка>

СЦЕНАРИЙ (по битам, ~20 сек):
• 0:03 — <реплика> | на экране: <короткий текст>
• 0:07 — <реплика> | на экране: <…>
• … (3–5 битов)

ПЕЙОФФ: <одна строка вывода>
CTA: <одна строка>

ПОДПИСЬ (caption): <2–4 строки, голосом Олеси>
ХЭШТЕГИ: <5–8 релевантных>
КАДРЫ (идеи): <кратко: планы/ракурсы для съёмки, 2–4 пункта>
```

Keep it tight. Every line earns its place.
